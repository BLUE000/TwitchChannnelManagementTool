# TwitchChannelManagementTool 設定項目定義書 (ドラフト)

本ドキュメントは、TwitchChannelManagementToolにおける各機能の設定項目を整理・定義するものです。基本設計（画面設計・DB設計）の基礎として活用します。

---

## 1. 接続・共通設定 (Connection & Common)

| 設定項目ID | 設定名 | 説明 | 管理単位 (案) | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| `common_channel_name` | Twitch チャンネル名 | 接続対象となるTwitchのチャンネル名。 | グローバル | |
| `common_oauth_token` | Twitch 認証トークン | APIやChat/EventSub接続用のOAuthトークン。 | グローバル | 難読化して保持 |
| `common_client_id` | Twitch クライアントID | Twitch APIアクセス用のクライアントID。 | グローバル | |
| `common_theme` | アプリケーションテーマ | ダークモード / ライトモード等の配色切替。 | グローバル | |

---

## 2. 時報設定 (Time Signals)

| :--- | :--- | :--- | :--- | :--- |

| `ts_text` | 読み上げテキスト | 指定時刻に読み上げる文字列内容。 | 個別 | |
| `ts_enabled` | 有効/無効 | この時報設定が現在有効かどうか。 | 個別 | |
| `ts_tts_engine` | 読み上げエンジン | 棒読みちゃん / VOICEVOX の選択。 | 個別/グローバル | |
| `ts_voice_type` | 声の種類 | 読み上げに使用するキャラ/話者ID。 | 個別/グローバル | |
| `ts_speed` | 読み上げ速度 | 読み上げ時の発話スピード。 | 個別/グローバル | |
| `ts_pitch` | 読み上げピッチ | 読み上げ時の声の高さ。 | 個別/グローバル | |
| `ts_volume` | 読み上げ音量 | 読み上げ時の音量。 | 個別/グローバル | |
| `ts_chat_notify` | チャット通知有無 | 時報と同時にチャットに通知文を投稿するか。 | 個別 | |
| `ts_chat_text` | チャット投稿テキスト | 同時チャット投稿時のメッセージ内容。 | 個別 | |

## 3. コメント表示・読み上げ設定 (Comments & TTS)

| 設定項目ID | 設定名 | 説明 | 管理単位 (案) | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| `comment_overlay_port` | オーバーレイ用ポート | OBS等に表示するHTMLを提供するHTTPサーバーのポート。 | グローバル | 既定: 8081 |
| `comment_overlay_style` | 表示スタイル | 標準スクロール / テロップ / 物理アバターなどの切り替え。 | グローバル | |
| `comment_font_size` | フォントサイズ | オーバーレイ表示時の文字の大きさ。 | グローバル | |
| `comment_font_color` | 文字色 | オーバーレイ表示時の文字の色。 | グローバル | |
| `comment_opacity` | エフェクト不透明度 | アバターやコメントの表示・消滅エフェクトの不透明度。 | グローバル | |
| `comment_show_avatar` | アバター表示有無 | コメントと同時に発言者のアバター画像を表示するかどうか。 | グローバル | |
| `comment_show_badges` | バッジ表示有無 | コメントと同時にバッジ（サブスク、モデレーター等）を表示するかどうか。 | グローバル | |
| `comment_show_emotes` | スタンプ表示有無 | コメント内のスタンプ（エモート）を画像として表示するかどうか。 | グローバル | |
| `comment_exclude_users` | 表示除外ユーザーリスト | 画面にコメントを表示したくないユーザーのリスト。 | グローバル/個別 | |
| `tts_enabled` | 読み上げ有効化 | コメント受信時の自動読み上げのON/OFF。 | グローバル | |
| `tts_bouyomi_host` | 棒読みちゃんホスト | 棒読みちゃんの接続先ホスト名/IP。 | グローバル | 既定: localhost |
| `tts_bouyomi_port` | 棒読みちゃんポート | 棒読みちゃんの接続先ポート番号。 | グローバル | 既定: 50001 |
| `tts_voicevox_host` | VOICEVOXホスト | VOICEVOXの接続先ホスト名/IP。 | グローバル | 既定: localhost |
| `tts_voicevox_port` | VOICEVOXポート | VOICEVOXの接続先ポート番号。 | グローバル | 既定: 50021 |
| `tts_user_voice_map` | ユーザー個別声割り当て | 特定の視聴者IDに対して特定の声（話者ID）を固定する設定。 | 個別 (ユーザーごと) | リスト形式で管理 |
| `tts_max_length` | 読み上げ最大文字数 | 読み上げるコメントの文字数の上限値（超過分はカットまたは「以下略」）。 | グローバル | |
| `tts_exclude_users` | 読み上げ除外ユーザーリスト | 読み上げ対象から除外したいユーザーのリスト。 | グローバル/個別 | |

## 4. チャンネルポイント演出設定 (Channel Points)

| :--- | :--- | :--- | :--- | :--- |
| `cp_reward_id` | 対象カスタム報酬ID | Twitch APIから取得したカスタム報酬の識別子 (UUID)。 | 個別 | |
| `cp_reward_name` | 対象カスタム報酬名 | Twitchダッシュボードで設定されたカスタム報酬の名称。 | 個別 | |
| `cp_image_path` | 演出用画像パス | 画面上に表示する画像・アニメーションGIFのファイルパス。 | 個別 | |
| `cp_video_path` | 演出用動画パス | 画面上に再生する動画ファイル（mp4, webm等）のパス。 | 個別 | |
| `cp_sound_path` | 演出用音声パス | 引き換え時に再生する効果音ファイルパス。 | 個別 | |
| `cp_volume` | 演出音量 | 演出効果音の再生ボリューム。 | 個別 | |
| `cp_duration` | 表示時間 | 画面上に演出を表示し続ける秒数。 | 個別 | |
| `cp_animation_type` | アニメーション種別 | フェードイン、バウンド、スライドなどの動き。 | 個別 | |
| `cp_text_enabled` | テキスト表示有無 | リスナーの入力やシステムメッセージをテキストとして表示するか。 | 個別 | |
| `cp_text_format` | テキスト表示形式 | テキスト表示時の文字列フォーマット（例：「@{user}が引き換えました！」）。 | 個別 | |
| `cp_pos_x` | 表示X座標 | 演出表示領域のX座標（またはパーセンテージ）。 | 個別 | 視覚的エディタから保存 |
| `cp_pos_y` | 表示Y座標 | 演出表示領域のY座標（またはパーセンテージ）。 | 個別 | 視覚的エディタから保存 |
| `cp_width` | 表示幅 | 演出表示領域の幅（またはパーセンテージ）。 | 個別 | 視覚的エディタから保存 |
| `cp_height` | 表示高さ | 演出表示領域の高さ（またはパーセンテージ）。 | 個別 | 視覚的エディタから保存 |

## 5. フォロワー管理設定 (Followers)

| 設定項目ID | 設定名 | 説明 | 管理単位 (案) | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| `follower_notify_enabled` | 新規フォロー通知ON/OFF | 新しいフォロワーが現れたときにチャットに通知を送るか。 | グローバル | |
| `follower_notify_text` | フォロー通知テキスト | チャットに送る通知文（例：「@{user}さんフォローありがとう！」）。 | グローバル | `{user}` などの変数をサポート |
| `follower_goal_value` | 目標フォロワー数 | 配信画面上に目標数と現在の割合を表示するための目標数値。 | グローバル | |
